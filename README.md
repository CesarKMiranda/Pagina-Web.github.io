# MI PRIMER PROYECTO EN GITHUB


## PROYECTO "CAFETERIA KVNCHI"

Sean bienvenidos al repositorio de mi primera incursión en el desarrollo web

Este documento sirve como guía para entender, configurar y ejecutar la aplicación principal

## VISION DEL PROYECTO

Este proyecto consiste en una página web estática simple, creada con el objetivo fundamental de aprender y aplicar los conceptos básicos de HTML5

Este proyecto representa un momento muy importante, ya que es mi primer documento HTML escrito y estructurado con ayuda de mi sensei, si bien la funcionalidad es limitada, el enfoque principal ha sido asegurar una estructura semántica clara, siguiendo las mejores prácticas y consejos de mi sensei.

## TECNOLOGIAS Y VERSION

#### ENFOQUE EN HTML5 BASICO

Se ha utilizado exclusivamente código HTML5 plano para la estructura. Esto significa que los componentes se basan en etiquetas fundamentales como:

Estructura Base: <!DOCTYPE html>, <html>, <head>, <body>.

Contenido: <h1>, <h2>, <p>, <a> (enlaces), <img> (imágenes).

Listas: <ul> y <li>.

Nota Importante: Este proyecto no incluye CSS (para estilos) ni JavaScript (para interactividad), ya que el objetivo era dominar primero la estructura pura de HTML

### Instrucciones de uso

Debido a que se trata de una página web estática y autocontenida, la configuración y el uso son muy sencillos

1. Requisitos

No se requiere software especializado más allá de: un navegador web (Chrome, Firefox, Edge, Safari, etc.)

2. Ejecución

Sigue estos pasos para ver la página en acción:

- Localiza el Archivo: Asegúrate de que tienes el archivo principal del proyecto llamado index.html guardado en una ubicación conocida de tu ordenador

- Abre con el Navegador: Haz doble clic en el archivo index.html o bien accede a través del vinculo https://cesarkmiranda.github.io/Pagina-Web.github.io/

- El navegador abrirá automáticamente el documento y lo renderizará
